## Flutter command >> 
- Follow dart package naming convention with lowercase_and_underscore, az09 and name should not start with digits or reserved word.
- `` mkdir flutterproject ``
- `` flutter create app_name ``
- `` cd app_name ``
- `` flutter doctor ``
- `` flutter run ``
- `` flutter emulators ``
- `` flutter devices ``
- `` flutter pub get ``
- `` flutter pub upgrade ``
- `` flutter clean ``
- `` flutter --version ``
